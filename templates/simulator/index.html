WELCOME TO THIS CALL CENTER SIMULATION APPLICATION!

How does it work?

  1. Please have all simulation parameters ready or use the default ones before proceeding
  2. Press "Enter" to input your variables
  3. Check and confirm your input
  4. Press "Run simulation" to start the simulation
  5. Please wait until the simulation is completed and then download the results

About

This project is an open-source call center simulation application primarily designed as a desktop app. It allows users to input various parameters and simulate a call center environment to test and analyze different scenarios. However, due to current server limitations, some inputs may be restricted. We welcome any contributions or feedback on the project. If you are interested in contributing or providing feedback, please contact yg2486@columbia.edu

Please note that this is a simulation application and the results should be used for reference purpose only
